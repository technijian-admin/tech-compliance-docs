## Access Control
- [ ] User access review report (all systems)
- [ ] Terminated user access removal confirmations
- [ ] MFA enrollment status report
- [ ] Privileged account inventory
- [ ] Role-based access matrix (current)

## Asset Management
- [ ] Hardware asset inventory export
- [ ] Software license inventory
- [ ] Asset disposal/decommission records (if any)
- [ ] Mobile device management (MDM) report

## Backup & Recovery
- [ ] Backup job success/failure logs (past 90 days)
- [ ] Restore test results (at least 1 per quarter)
- [ ] Backup retention policy confirmation
- [ ] Offsite/cloud backup verification

## Change Management
- [ ] Change request tickets (sample of 5-10)
- [ ] Change advisory board (CAB) meeting minutes
- [ ] Emergency change records (if any)

## Configuration Management
- [ ] Server hardening baseline compliance report
- [ ] GPO/policy configuration exports
- [ ] Cloud configuration review (if applicable)

## Data Protection
- [ ] Encryption-at-rest verification (databases, file shares)
- [ ] Encryption-in-transit verification (TLS configs)
- [ ] DLP policy status and alert summary
- [ ] Data classification inventory (if updated)

## Endpoint Security
- [ ] AV/EDR deployment coverage report
- [ ] AV/EDR detection and quarantine summary
- [ ] Endpoint compliance status (OS versions, encryption)

## Incident Response
- [ ] Incident tickets from the quarter (if any)
- [ ] IR tabletop exercise results (if conducted)
- [ ] Post-incident review reports (if any)

## Logging & Monitoring
- [ ] SIEM/log aggregation uptime report
- [ ] Alert review and triage summary
- [ ] Log retention verification
- [ ] Sample of investigated alerts (5-10)

## Network Security
- [ ] Firewall rule review documentation
- [ ] Vulnerability scan results (internal + external)
- [ ] Penetration test report (if conducted this quarter)
- [ ] Network diagram (if updated)

## Patch Management
- [ ] Patch compliance report (servers)
- [ ] Patch compliance report (workstations)
- [ ] Critical/high patch remediation timelines

## Physical Security
- [ ] Facility access logs (badge reports)
- [ ] Visitor logs (if applicable)
- [ ] Security camera status verification

## Risk Management
- [ ] Risk register (current version)
- [ ] New risks identified this quarter
- [ ] Risk treatment plan updates

## Security Awareness
- [ ] Training completion report
- [ ] Phishing simulation results
- [ ] New hire security onboarding records

## Vendor Management
- [ ] Vendor risk assessment updates (if any)
- [ ] BAA/DPA inventory (current)
- [ ] Vendor security review results (if any)
- [ ] SLA compliance reports from critical vendors
